Une offre logicielle maintenue par un fournisseur tiers.

<!--more-->

Parmi les exemples de services gérés, on trouve AWS EC2, Azure SQL Database et
GCP Pub/Sub, mais il peut s'agir de tout logiciel accessible et utilisé par une application.
