Un plugin de volume permet d'intégrer du stockage dans un {{< glossary_tooltip text="Pod" term_id="pod" >}}.

<!--more--> 

Un plugin de volume permet de connecter et de monter des volumes de stockage pour un {{< glossary_tooltip text="Pod" term_id="pod" >}}. Les plugins de volume peuvent être _in tree_ ou _out of tree_. Les plugins _in tree_ font partie du dépôt de code Kubernetes et suivent son cycle de publication. Les plugins _out of tree_ sont développés de manière indépendante.
