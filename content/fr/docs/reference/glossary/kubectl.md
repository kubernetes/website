Outil en ligne de commande pour communiquer avec le {{< glossary_tooltip text="plan de contrôle" term_id="control-plane" >}} d’un cluster Kubernetes via l’API Kubernetes.

<!--more--> 

Vous pouvez utiliser `kubectl` pour créer, inspecter, mettre à jour et supprimer des objets Kubernetes.

<!-- note de localisation : le reste peut être omis -->
En anglais, `kubectl` se prononce officiellement /kjuːb/ /kənˈtɹəʊl/ (comme "cube control").
