Le dockershim est un composant de Kubernetes version 1.23 et antérieure. Il permet au {{< glossary_tooltip text="kubelet" term_id="kubelet" >}} de communiquer avec {{< glossary_tooltip text="Docker Engine" term_id="docker" >}}.

<!--more-->

À partir de la version 1.24, dockershim a été supprimé de Kubernetes. Pour plus d’informations, voir [FAQ Dockershim](/dockershim).
