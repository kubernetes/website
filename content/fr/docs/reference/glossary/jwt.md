Un moyen de représenter des assertions à transférer entre deux parties.

<!--more-->

Les JWT peuvent être signés numériquement et chiffrés. Kubernetes utilise les JWT comme
tokens d’authentification pour vérifier l’identité des entités souhaitant effectuer
des actions dans un cluster.
