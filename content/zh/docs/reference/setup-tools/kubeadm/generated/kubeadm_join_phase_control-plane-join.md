### 概要


<!--
Join a machine as a control plane instance
-->
添加作为控制平面实例的机器

```
kubeadm join phase control-plane-join [flags]
```

<!--
### Examples
-->
### 示例

<!--
```
# Joins a machine as a control plane instance
kubeadm join phase control-plane-join all
```
-->
```
# 将机器作为控制平面实例加入
kubeadm join phase control-plane-join all
```
