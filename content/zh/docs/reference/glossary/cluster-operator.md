他们的主要责任是保持集群正常运行，可能需要进行周期性的维护和升级活动。<br>

**注意：** 集群操作者不同于[操作者模式（Operator Pattern）](https://www.openshift.com/learn/topics/operators)，操作者模式是用来扩展 Kubernetes API 的。
