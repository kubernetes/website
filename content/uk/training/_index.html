Побудуйте свою карʼєру з хмарними технологіями

Kubernetes є основою руху в хмарні технології. Навчання та сертифікація від Linux Foundation та наших партнерів дозволяють інвестувати у вашу карʼєру, вивчити Kubernetes та зробити ваші проєкти з хмарними технологіями успішними.

Отримайте сертифікат Kubernetes

Kubernetes and Cloud Native Associate (KCNA)

Іспит KCNA підтверджує базові знання та навички користувача у сфері Kubernetes і ширшої екосистеми хмарних технологій.

Сертифікат KCNA підтверджує концептуальні знання всієї екосистеми хмарних технологій, зокрема з акцентом на Kubernetes.


Перейти до сертифікації
Kubernetes and Cloud Native Security Associate (KCSA)

KCSA — це попередня професійна сертифікація, призначена для кандидатів, зацікавлених у переході на професійний рівень через демонстрацію розуміння фундаментальних знань та навичок технологій безпеки в хмарній екосистемі.

Сертифікат KCSA підтверджує розуміння базової конфігурації безпеки кластерів Kubernetes для досягнення поставлених цілей.


Перейти до сертифікації
Certified Kubernetes Application Developer (CKAD)

Іспит Certified Kubernetes Application Developer засвідчує, що користувачі можуть проєктувати, створювати, налаштовувати та розгортати хмарні застосунки для Kubernetes.

Сертіфіковані CKAD спеціалісти можуть визначати ресурси застосунків та використовувати основні примітиви для побудови, моніторингу та усунення несправностей масштабованих застосунків та інструментів у Kubernetes.


Перейти до сертифікації
Certified Kubernetes Administrator (CKA)

Програма Certified Kubernetes Administrator (CKA) гарантує, що власники сертифікатів CKA володіють навичками, знаннями та компетенцією для виконання обовʼязків адміністраторів Kubernetes.

Сертифікований адміністратор Kubernetes демонструє вміння виконувати базову інсталяцію, а також налаштовувати та керувати кластерами Kubernetes операційного рівня.


Перейти до сертифікації
Certified Kubernetes Security Specialist (CKS)

Програма Certified Kubernetes Security Specialist забезпечує впевненість у тому, що власник сертифікату володіє широким спектром найкращих практик і є компетентним у цій сфері. Сертифікація CKS охоплює навички захисту контейнерних застосунків та платформ Kubernetes під час створення, розгортання та експлуатації.

Кандидати на CKS повинні мати чинний сертифікат сертифікованого адміністратора Kubernetes (CKA), щоб продемонструвати, що вони володіють достатніми знаннями Kubernetes, перш ніж складати іспит на CKS.


Перейти до сертифікації

Програма Kubestronaut: Підвищуйте свою експертизу в Kubernetes

Програма CNCF Kubestronaut відзначає та нагороджує виняткових лідерів спільноти, які продемонстрували глибоку прихильність до оволодіння Kubernetes. Ця престижна ініціатива відзначає людей, які постійно інвестують у своє навчання та досягли найвищого рівня майстерності. Щоб отримати почесне звання Kubestronaut, люди повинні успішно отримати та підтримувати всі пʼять сертифікатів CNCF Kubernetes: Сертифікований адміністратор Kubernetes (CKA), Сертифікований розробник застосунків Kubernetes (CKAD), Сертифікований спеціаліст з безпеки Kubernetes (CKS), Kubernetes та Cloud Native Associate (KCNA) та Kubernetes та Cloud Security Associate (KCSA).
