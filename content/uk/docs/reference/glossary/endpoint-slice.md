EndpointSlices відстежують IP-адреси Podʼів з відповідними {{< glossary_tooltip text="селекторами" term_id="selector" >}} Service.

<!--more-->

EndpointSlices можуть бути налаштовані вручну для {{< glossary_tooltip text="Services" term_id="service" >}} без вказання селекторів.
