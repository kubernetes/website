## {{% heading "synopsis" %}}

kubectl керує менеджером кластерів Kubernetes.

Додаткову інформацію можна знайти в розділі [Інструмент командного рядка](/docs/reference/kubectl/) (`kubectl`).

```shell
kubectl [flags]
```
