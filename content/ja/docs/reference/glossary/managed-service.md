サードパーティのプロバイダーによって保守されるソフトウェアの提供形態です。

<!--more--> 

マネージドサービスの例としては、AWS EC2、Azure SQL Database、GCP Pub/Subなどがありますが、アプリケーションで使用できるあらゆるソフトウェアの提供形態が該当します。
