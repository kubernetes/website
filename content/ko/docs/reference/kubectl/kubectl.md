## {{% heading "synopsis" %}}


kubectl은 쿠버네티스 클러스터 관리자를 제어한다.

 자세한 정보는 https://kubernetes.io/docs/reference/kubectl/overview/ 에서 확인한다.

```
kubectl [flags]
```
