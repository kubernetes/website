<!--more-->
Kubelet이 설정에서 스태틱 파드를 찾으면, 자동으로 쿠버네티스
API 서버에 파드 객체 생성을 시도한다. 이렇게 생성된 파드를
API 서버에서 확인할 수는 있지만, API 서버를 통해 제어할 수는 없다.

(예를 들어, 미러 파드를 제거하더라도 kubelet 데몬이 해당 파드를 멈추지 않는다.)
