Một họ các loại API dùng để mô hình hóa mạng dịch vụ trong Kubernetes.

<!--more-->

Gateway API cung cấp một họ các loại API có thể mở rộng, định hướng vai trò và nhận diện giao thức để mô hình hóa mạng dịch vụ trong Kubernetes.
