 Một đối tượng API quản lý truy cập bên ngoài đến các dịch vụ trong cluster, thường là HTTP.

<!--more--> 

Ingress có thể cung cấp các tính năng như cân bằng tải, SSL termination và virtual hosting dựa trên tên miền.
