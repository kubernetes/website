Một API mở rộng trước đây cho phép ứng dụng chạy trong cluster Kubernetes dễ dàng sử dụng các dịch vụ phần mềm được quản lý bên ngoài, chẳng hạn dịch vụ kho dữ liệu do nhà cung cấp cloud cung cấp.

<!--more-->

API này cung cấp cách liệt kê, cấp phát và liên kết với {{< glossary_tooltip text="Managed Services" term_id="managed-service" >}} bên ngoài mà không cần hiểu chi tiết cách các dịch vụ đó được tạo hoặc quản lý.
