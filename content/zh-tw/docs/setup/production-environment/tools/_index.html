--- title: 使用部署工具安裝 Kubernetes weight: 30 no_list: true --- 有許多方法與工具可以用來建立你自己的生產環境 Kubernetes 叢集。例如： - [kubeadm](/docs/setup/production-environment/tools/kubeadm/) - [Cluster API](https://cluster-api.sigs.k8s.io/)：Kubernetes 的一個子專案，專注於提供宣告式 API 與工具，以簡化多個 Kubernetes 叢集的佈建、升級與運作。 - [kops](https://kops.sigs.k8s.io/)：一個自動化的叢集佈建工具。關於教學、最佳實踐、設定選項以及如何與社群聯繫， 請參考 [`kOps` 官方網站](https://kops.sigs.k8s.io/) 以取得更多資訊。 - [kubespray](https://kubespray.io/)： 由 [Ansible](https://docs.ansible.com/) playbooks、 [inventory](https://github.com/kubernetes-sigs/kubespray/blob/master/docs/ansible/inventory.md)、 佈建工具以及領域知識所組成，用於通用作業系統與 Kubernetes 叢集的設定管理。 你也可以透過 Slack 頻道 [#kubespray](https://kubernetes.slack.com/messages/kubespray/) 與社群聯繫。
