- **acquireTime** (MicroTime)

  acquireTime 是當前租約被獲取的時間。
  
  <a name="MicroTime"></a>
  **MicroTime 是微秒級精確時間的版本。**

- **holderIdentity** (string)

  holderIdentity 包含當前租約持有人的身份。
  如果使用協調領導者選舉（Coordinated Leader Election），
  則持有人身份必須等於選舉出的 LeaseCandidate.metadata.name 字段。

<!--
- **leaseDurationSeconds** (int32)

  leaseDurationSeconds is a duration that candidates for a lease need to wait to force acquire it. This is measured against the time of last observed renewTime.

- **leaseTransitions** (int32)

  leaseTransitions is the number of transitions of a lease between holders.

- **preferredHolder** (string)

  PreferredHolder signals to a lease holder that the lease has a more optimal holder and should be given up. This field can only be set if Strategy is also set.
-->
- **leaseDurationSeconds** (int32)

  leaseDurationSeconds 是租約候選人需要等待強制獲取租約的持續時間。
  這是相對於上次觀察到的 renewTime 的度量。

- **leaseTransitions** (int32)

  leaseTransitions 是租約持有人之間的轉換次數。

- **preferredHolder** (string)

  preferredHolder 向租約持有人發出信號，提示此租約出現一個更優的持有人且應該被放棄。
  此字段僅在設置了 `strategy` 時才能被設置。

<!--
- **renewTime** (MicroTime)

  renewTime is a time when the current holder of a lease has last updated the lease.

  <a name="MicroTime"></a>
  *MicroTime is version of Time with microsecond level precision.*

- **strategy** (string)

  Strategy indicates the strategy for picking the leader for coordinated leader election. If the field is not specified, there is no active coordination for this lease. (Alpha) Using this field requires the CoordinatedLeaderElection feature gate to be enabled.
-->
- **renewTime** (MicroTime)

  renewTime 是當前租約持有人上次更新租約的時間。

  <a name="MicroTime"></a>
  **MicroTime 是具有微秒級精度的時間版本。**

- **strategy**（字符串）

  strategy 指示爲協調領導者選舉選擇領導者的策略。
  如果此字段未被指定，則此租約沒有主動的協調。
  （Alpha）使用此字段需要啓用 CoordinatedLeaderElection 特性門控。
