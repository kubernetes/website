獲取 Kubernetes 認證

Kubernetes and Cloud Native Associate (KCNA)

Kubernetes 和雲原生夥伴（KCNA）考試展示了使用者在 Kubernetes 和更廣泛的雲原生生態系統中的基本知識和技能。

獲得認證的 KCNA 將確認整個雲原生生態系統的概念性知識，特別是 Kubernetes。


前往認證
Kubernetes 和雲原生安全助理 (KCSA)

KCSA 是一項專業前認證，專爲有興趣通過對雲原生生態系統中安全技術的基礎知識和技能的瞭解而提升到專業水平的考生而設計。

經過認證的 KCSA 將確認對 Kubernetes 叢集的基線安全設定的理解，以滿足合規性目標。


前往認證
Certified Kubernetes Application Developer (CKAD)

CKAD 考試證明使用者可以爲 Kubernetes 設計、構建、設定和發佈雲原生應用。

通過 CKAD 認證的人員能夠在 Kubernetes 中定義軟件資源並且使用核心組件來搭建、監控、檢查可伸縮應用


前往認證
Certified Kubernetes Administrator (CKA)

CKA 認證確保 CKA 認證人員具有履行 Kubernetes 管理員職責的技能、知識和能力。

通過認證的 Kubernetes 管理員展示了包括基礎安裝和設定、管理產品級別的 Kubernetes 叢集方面的能力。


前往認證
Kubernetes 認證安全專家 (CKS)

Kubernetes 認證安全專家項目確保證書持有人員容易勝任各種最佳實踐。CKS 認證包括在構建、部署和運行時確保基於容器的應用和 Kubernetes 平臺安全的技能。

參加 CKS 的候選人必須持有現行的 Kubernetes 認證管理員（CKA）證書，以證明他們具有足夠的 Kubernetes 專業知識。


前往認證

Kubestronaut 計劃：提升你的 Kubernetes 專業技能

CNCF 的 Kubestronaut 計劃旨在表彰和認可熟練掌握 Kubernetes 且表現卓越的社區領導者。 這一享有盛譽的計劃專注於那些持續投入學習並達到最高專業水平的個人。 要獲得 Kubestronaut 的榮譽稱號，個人必須成功獲得並保持 CNCF 的五項 Kubernetes 認證： Certified Kubernetes Administrator (CKA)、Certified Kubernetes Application Developer (CKAD)、 Certified Kubernetes Security Specialist (CKS)、Kubernetes and Cloud Native Associate (KCNA) 以及 Kubernetes and Cloud Security Associate (KCSA)。
