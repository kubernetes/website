El dockershim es un componente de Kubernetes de la versión 1.23 y anteriores. Permite que el {{< glossary_tooltip text="kubelet" term_id="kubelet" >}}
se comunique con {{< glossary_tooltip text="Docker Engine" term_id="docker" >}}.

<!--more-->
A partir de la versión 1.24, dockershim ha sido eliminado de Kubernetes. Para más información, consulta las [Preguntas frecuentes sobre Dockershim](/dockershim).
