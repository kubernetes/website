允许 kubelet 为节点上的 Kubernetes 工作负载分配交换内存。
必须将 `KubeletConfiguration.failSwapOn` 设置为 false 才能使用此能力。
更多细节请参见[交换内存](/zh-cn/docs/concepts/architecture/nodes/#swap-memory)。
