## {{% heading "synopsis" %}}

<!--
Mark node as unschedulable.
-->
将节点标记为不可调度的。

```bash
kubectl cordon NODE
```

## {{% heading "examples" %}}

<!--
```
  # Mark node "foo" as unschedulable
  kubectl cordon foo
```
-->
```bash
# 将节点 "foo" 标记为不可调度的
kubectl cordon foo
```
