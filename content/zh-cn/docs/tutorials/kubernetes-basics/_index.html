Kubernetes 基础知识

本教程介绍了 Kubernetes 集群编排系统的基础知识。每个模块包含关于 Kubernetes 主要特性和概念的一些背景信息， 还包括一个在线互动教程。

使用互动教程，你可以学习：

  • 在集群上部署容器化应用。
  • 弹性部署。
  • 使用新的软件版本，更新容器化应用。
  • 调试容器化应用。

Kubernetes 可以为你做些什么?

通过现代的 Web 服务，用户希望应用能够 24/7 全天候使用，开发人员希望每天可以多次发布部署新版本的应用。 容器化可以帮助软件包达成这些目标，使应用能够以简单快速的方式发布和更新，而无需停机。Kubernetes 帮助你确保这些容器化的应用在你想要的时间和地点运行，并帮助应用找到它们需要的资源和工具。 Kubernetes 是一个可用于生产的开源平台，根据 Google 容器集群方面积累的经验，以及来自社区的最佳实践而设计。
