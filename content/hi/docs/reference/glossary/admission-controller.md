कोड का एक भाग जो ऑब्जेक्ट के स्थायी रूप से संग्रहीत होने से पहले कुबेरनेट्स API सर्वर को भेजे गए अनुरोधों को इंटरसेप्ट करता है।

<!--more-->

एडमिशन कंट्रोलर कुबेरनेट्स API सर्वर के लिए कॉन्फ़िगर किए जा सकते हैं और ये "प्रमाणीकरण" (validating), "परिवर्तन" (mutating), या दोनों हो सकते हैं। कोई भी एडमिशन कंट्रोलर अनुरोध को अस्वीकार कर सकता है। परिवर्तन कंट्रोलर उन ऑब्जेक्ट्स को संशोधित कर सकते हैं जिन्हें वे स्वीकार करते हैं; प्रमाणीकरण कंट्रोलर ऐसा नहीं कर सकते।
