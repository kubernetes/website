 مجموعه‌ای از گونه‌های API برای مدل‌سازی شبکه‌سازی سرویس در کوبرنتیز.

<!--more--> 

Gateway API مجموعه‌ای از گونه‌های API قابل‌گسترش، نقش‌محور و آگاه از پروتکل را برای مدل‌سازی شبکه‌سازی سرویس در کوبرنتیز فراهم می‌کند.
