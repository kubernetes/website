 خانواده‌ای از انواع API برای مدل‌سازی شبکه سرویس در کوبرنتیز.

<!--more--> 

Gateway API مجموعه‌ای از انواع API توسعه‌پذیر، نقش‌گرا و آگاه از پروتکل را برای مدل‌سازی شبکه‌بندی سرویس در کوبرنتیز فراهم می‌کند.
