 یک شیء API که دسترسی خارجی به سرویس‌ها را در یک کلاستر، معمولاً HTTP، مدیریت می‌کند.

<!--more--> 

Ingress ممکن است متعادل‌سازی بار، خاتمه SSL و میزبانی مجازی مبتنی بر نام را ارائه دهد.
