 یک شیء API که دسترسی خارجی به سرویس‌های یک خوشه، معمولاً HTTP، را مدیریت می‌کند.

<!--more--> 

اینگرس ممکن است توازن بار، پایان‌دهی SSL و میزبانی مجازی مبتنی بر نام را فراهم کند.
