docker (به طور خاص، موتور docker) یک فناوری نرم‌افزاری است که مجازی‌سازی در سطح سیستم عامل را ارائه می‌دهد و با نام {{< glossary_tooltip text="containers" term_id="container" >}} نیز شناخته می‌شود.

<!--more-->

docker از ویژگی‌های جداسازی منابع هسته(kernel) لینوکس مانند cgroups و kernel namespaces و یک سیستم فایل با قابلیت ادغام مانند OverlayFS و موارد دیگر استفاده می‌کند تا به کانتینرهای مستقل اجازه دهد تا در یک نمونه لینوکس واحد اجرا شوند و از سربار راه‌اندازی و نگهداری ماشین‌های مجازی (VM) جلوگیری شود.
