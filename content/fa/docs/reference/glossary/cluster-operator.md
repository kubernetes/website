 شخصی که خوشه ها را پیکربندی، کنترل و نظارت می‌کند.

<!--more--> 

مسئولیت اصلی آنها فعال و در حال اجرا نگه داشتن یک خوشه است که ممکن است شامل فعالیت‌های نگهداری یا ارتقاء دوره‌ای باشد.

{{< note >}}
عملگرهای خوشه با الگوی عملگر (/docs/concepts/extend-kubernetes/operator/) که API کوبرنتیز را گسترش می‌دهد، متفاوت هستند.
{{< /note >}}
