وسیله‌ای برای نمایش منبع منحصر به فرد API کوبرنتیز.

<!--more-->

منابع نسخه گروهی (GVR) گروه API، نسخه API و منبع (نام نوع شیء همانطور که در URI نشان داده می‌شود) مرتبط با دسترسی به یک شناسه خاص از شیء در کوبرنتیز را مشخص می‌کنند.
GVRها به شما امکان می‌دهند اشیاء مختلف کوبرنتیز را تعریف و از هم متمایز کنید و روشی برای دسترسی به اشیاء مشخص کنید که حتی با تغییر APIها نیز پایدار باشد.
