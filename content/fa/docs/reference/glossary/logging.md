 لاگ‌ها فهرست رویدادهایی هستند که توسط {{< glossary_tooltip text="کلاستر" term_id="cluster" >}} یا برنامه ثبت می‌شوند.

<!--more--> 

لاگ‌های برنامه و سیستم به شما کمک می‌کنند بفهمید داخل کلاستر چه می‌گذرد. لاگ‌ها به‌ویژه برای دیباگ مشکلات و پایش فعالیت کلاستر مفید هستند.
