 راهی برای نمایش یک برنامه در حال اجرا روی یک 
{{< glossary_tooltip text="Pod" term_id="pod" >}} یا تعداد بیش‌تر Pod در برنامه شما.

<!--more-->
مجموعه پادهای مورد هدف یک سرویس (معمولاً) توسط یک
{{< glossary_tooltip text="selector" term_id="selector" >}} تعیین می‌شود. 
اگر پادهای بیشتری اضافه یا حذف شوند، مجموعه پادهای منطبق با selector تغییر خواهد کرد. 
سرویس اطمینان حاصل می‌کند که ترافیک شبکه می‌تواند به مجموعه پادهای فعلی برای بار کاری هدایت شود.

سرویس‌های کوبرنتیز یا از شبکه IP (IPv4، IPv6 یا هر دو) استفاده می‌کنند، یا به یک نام خارجی در سیستم نام دامنه (DNS) ارجاع می‌دهند.

موجودیت سرویس، سازوکارهای دیگری مانند Ingress و Gateway را فعال می‌کند.
