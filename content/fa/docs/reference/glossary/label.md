برچسب‌ها جفت‌های کلید/مقدار هستند که به اشیائی مانند {{< glossary_tooltip text="Pods" term_id="pod" >}} متصل می‌شوند. از آن‌ها برای سازمان‌دهی و انتخاب زیرمجموعه‌ای از اشیاء استفاده می‌شود.
