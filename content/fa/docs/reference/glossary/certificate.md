 یک فایل رمزنگاری‌شده‌ی امن که برای اعتبارسنجی دسترسی به کلاستر کوبرنتیز استفاده می‌شود.

<!--more--> 

گواهی‌ها به برنامه های درون یک کلاستر کوبرنتیز امکان دسترسی ایمن به API Kubernetes را می‌دهند. گواهی‌ها تأیید می‌کنند که کلاینت‌ها مجاز به دسترسی به API هستند.
