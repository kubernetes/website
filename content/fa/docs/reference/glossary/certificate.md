 یک فایل رمزنگاری‌شده‌ی امن که برای اعتبارسنجی دسترسی به کلاستر کوبرنتیز استفاده می‌شود.

<!--more--> 

گواهینامه‌ها به برنامه‌های درون یک کلاستر کوبرنتیز امکان دسترسی ایمن به API کوبرنتیز را می‌دهند. گواهینامه‌ها تأیید می‌کنند که کلاینت‌ها مجاز به دسترسی به API هستند.
