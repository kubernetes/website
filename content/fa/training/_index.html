شغل Cloud Native خود را بسازید

کوبرنتیز در هسته جنبش Cloud Native قرار دارد. آموزش و گواهینامه‌های بنیاد لینوکس و شرکای آموزشی ما به شما این امکان را می‌دهد که در حرفه خود سرمایه‌گذاری کنید، کوبرنتیز را بیاموزید و پروژه‌های Cloud Native خود را با موفقیت به انجام برسانید.

دریافت گواهینامه کوبرنتیز

همکار کوبرنتیز و Cloud Native (KCNA)

آزمون Kubernetes and Cloud Native Associate (KCNA) دانش و مهارت‌های بنیادی کاربر را در کوبرنتیز و بوم‌سازگان گسترده‌تر Cloud Native نشان می‌دهد.

یک KCNA دارای گواهینامه، دانش مفهومی از کل بوم‌سازگان Cloud Native، به ویژه با تمرکز بر کوبرنتیز را تأیید می‌کند.


برای دریافت گواهینامه کلیک کنید
همکار کوبرنتیز و امنیت Cloud Native (KCSA)

KCSA یک گواهینامه پیش‌حرفه‌ای است که برای داوطلبانی طراحی شده است که علاقه‌مند به پیشرفت به سطح حرفه‌ای از طریق درک اثبات‌شده از دانش و مهارت‌های بنیادی فناوری‌های امنیتی در بوم‌سازگان Cloud Native هستند.

یک KCSA دارای گواهینامه، درک پیکربندی امنیتی پایه خوشه‌های کوبرنتیز را برای دستیابی به اهداف انطباق تأیید می‌کند.


برای دریافت گواهینامه کلیک کنید
توسعه‌دهنده برنامه گواهی‌شده کوبرنتیز (CKAD)

آزمون Certified Kubernetes Application Developer گواهی می‌دهد که کاربران می‌توانند برنامه‌های Cloud Native را برای کوبرنتیز طراحی، ساخت، پیکربندی و ارائه کنند.

یک CKAD می‌تواند منابع برنامه را تعریف کند و از اصول اولیه اصلی برای ساخت، نظارت و عیب‌یابی برنامه‌ها و ابزارهای مقیاس‌پذیر در کوبرنتیز استفاده کند.


برای دریافت گواهینامه کلیک کنید
مدیر گواهی‌شده کوبرنتیز (CKA)

برنامه‌ی Certified Kubernetes Administrator (CKA) تضمین می‌کند که CKAها مهارت، دانش و شایستگی لازم برای انجام مسئولیت‌های مدیریت کوبرنتیز را دارند.

یک مدیر کوبرنتیز دارای گواهینامه، توانایی انجام نصب اولیه و همچنین پیکربندی و مدیریت خوشه‌های کوبرنتیز در محیط عملیاتی را دارد.


برای دریافت گواهینامه کلیک کنید
متخصص امنیت گواهی‌شده کوبرنتیز (CKS)

برنامه متخصص امنیت گواهی شده کوبرنتیز تضمین می‌کند که دارنده آن با طیف گسترده‌ای از بهترین شیوه‌ها راحت و ماهر است. گواهینامه CKS مهارت‌هایی را برای ایمن‌سازی برنامه‌های مبتنی بر کانتینر و پلتفرم‌های کوبرنتیز در طول ساخت، استقرار و زمان اجرا پوشش می‌دهد..

داوطلبان CKS باید دارای گواهینامه معتبر CKA باشند تا نشان دهند که قبل از شرکت در CKS، تخصص کافی در کوبرنتیز دارند.


برای دریافت گواهینامه کلیک کنید

برنامه کوبرنتیز: تخصص خود را در کوبرنتیز ارتقا دهید

برنامه کوبرنتیز متعلق به CNCF، رهبران استثنایی جامعه را که تعهد عمیقی به تسلط بر کوبرنتیز نشان داده‌اند، تجلیل و تقدیر می‌کند. این ابتکار معتبر، افرادی را که به طور مداوم در آموزش مداوم خود سرمایه‌گذاری کرده‌اند و به بالاترین سطوح مهارت دست یافته‌اند، برجسته می‌کند. برای کسب عنوان معتبر Kubestronaut، افراد باید با موفقیت هر پنج گواهینامه کوبرنتیز CNCF را کسب و حفظ کنند: مدیر گواهی‌شده کوبرنتیز (CKA)، توسعه‌دهنده برنامه گواهی‌شده کوبرنتیز (CKAD)، متخصص امنیت گواهی‌شده کوبرنتیز (CKS)، همکار کوبرنتیز و Cloud Native (KCNA) و همکار کوبرنتیز و امنیت Cloud Native (KCSA).

شرکای آموزشی کوبرنتیز

شبکه شرکای آموزشی کوبرنتیز ما، خدمات آموزشی برای کوبرنتیز و پروژه‌های Cloud Native ارائه می‌دهد.
